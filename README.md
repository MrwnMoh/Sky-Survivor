# 🚀 Sky Survivor (C# WinForms)

**Sky Survivor** is a 2D survival game built using **C# and .NET WinForms** — without any game engine.

🎮 **Gameplay**
Survive endless waves, collect coins, and use abilities like rockets to stay alive as long as possible.

## ✨ Features
- 🌊 Wave-based difficulty system  
- 🪙 Coin collection system  
- 🚀 Abilities with cooldowns  
- 💀 Game Over screen (Time, Coins, Wave)  
- 🎨 Custom animation system (Idle, Movement, Slide, Arrows)

## 🛠 Tech
- C#  
- .NET WinForms  
- OOP  
- Timers & Event-driven programming  

## 🎯 Learning
- Built a game from scratch (no engine)  
- Created animation system manually  
- Managed game logic & real-time interactions
